# Vendor App 🧰

The app that **service providers (vendors)** use.

## What vendors can do here
- Add and edit their services
- Accept or reject booking requests
- Manage their schedule
- View earnings and payments
- Receive notifications

## Folders
```
vendor-app/
├── screens/      → App pages (Dashboard, My Services, Bookings, etc.)
├── components/   → Reusable UI pieces
├── services/     → Code that talks to the backend (API calls)
└── assets/       → Images, icons, fonts
```

## Suggested Tech
React Native / Flutter
