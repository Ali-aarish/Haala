# Admin Panel 🖥️

The web dashboard that the **platform team** uses to run Haala.

## What admins can do here
- Manage customers and vendors
- Review and approve services
- Monitor bookings and payments
- Handle support and operational tasks
- View reports and platform activity

## Folders
```
admin-panel/
├── pages/        → Dashboard pages (Users, Services, Payments, etc.)
├── components/   → Reusable UI pieces (tables, charts, forms)
├── services/     → Code that talks to the backend (API calls)
└── assets/       → Images, icons, styles
```

## Suggested Tech
React (web)
