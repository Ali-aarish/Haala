# Customer App 📱

The app that **customers** use.

## What customers can do here
- Browse and search for services
- Book a service and pick a time
- Pay for bookings
- Track booking status
- Receive notifications

## Folders
```
customer-app/
├── screens/      → App pages (Home, Booking, Payment, etc.)
├── components/   → Reusable UI pieces (buttons, cards, etc.)
├── services/     → Code that talks to the backend (API calls)
└── assets/       → Images, icons, fonts
```

## Suggested Tech
React Native / Flutter
